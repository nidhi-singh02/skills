# Social metadata playbook (per-platform algorithm)

Write `<edit>/social_metadata.md` with sections for each platform the user wants. The
DURABLE rules are below; **trends move — web-search current best practices first** and
adjust counts/phrasing. Always tie titles/captions to the video's cold-open hook (every
platform now rewards retention/"quality CTR" — if the title overpromises vs the first 2s,
it gets demoted).

## YouTube (Shorts)
Algorithm: CTR + retention ("Quality CTR" — high click but low 15–30s retention is demoted).
- **Titles:** give ~6 ranked options. ≤~45 chars, DECLARATIVE (not questions), front-load,
  include a number or "I/you". Formulas: number, curiosity gap, transformation, warning,
  year+action, versus. Keep `#Shorts` in the description, not the title.
- **Title ≠ description first line (REQUIRED):** show them side by side. Title = the promise
  (swipe-stopper); first line = the specifics/keywords for search, zero word-for-word overlap.
- **Description:** hook line, then 2–4 lines of what it shows (keyword-rich), then links
  (mark `[add link]`), then a comment-bait question + follow CTA, then 3 hashtags.
- **Tags:** ~15–25 comma-separated (minor signal now; relevance > volume).
- **Hashtags:** 3 above the title (`#AI #Coding #Shorts`) + ≤~10 total.
- **Thumbnail:** 3–4 ideas, ≤4 words, expressive face, high contrast, brand accent.

## Instagram (Reels)
Algorithm: SHARES (DM sends weighted 3–5× over likes) + saves + watch time. Captions/on-screen
text are crawled like search; hashtags are a weak signal.
- **Caption:** hook first line (different from the on-screen title — don't repeat it). 2–3
  KEYWORD-RICH sentences up top (that's the discovery signal). CTAs target SAVE + SEND
  ("save this for later", "send to the friend who…"), then a comment question, then follow.
- **Hashtags:** 3–5 MAX. 6+ gets flagged low-intent and deprioritized. Niche > broad.
- **Add Topics:** set 2–3 native topics in the upload flow (2026 targeting).
- **Cover:** keep face + text in the CENTER safe zone (grid crops 4:5/1:1 — top ~15% /
  bottom ~20% get cut). Set a custom cover frame; don't let IG auto-pick.
- **Audio:** voiceover stays primary; optional trending track at ~5–10% under it.

## X (native video)
Algorithm: replies (author replying = the single biggest lever) > bookmarks (5× weight) >
reposts > likes. Upload NATIVE; <60s gets the biggest boost; 50%+ completion extends reach.
- **No links in the post body** — they cut reach 30–50%. Put links in the FIRST REPLY.
- **Post copy:** hook line + keyword-rich plain text (X ranks on semantics, not tags) +
  END ON A QUESTION to drive replies. Reply to everyone in the first 1–2 hrs. No
  engagement bait ("RT if…") — penalized.
- **Hashtags:** 0–1 only (multiple = −40%).
- **Cover/poster:** autoplays MUTED → first frame must stop the scroll; keep burned-in captions.
- Offer an optional 3–4 tweet THREAD version (more dwell time = more reach).

## TikTok (add when asked)
Algorithm: completion rate + rewatches + shares; on-screen text + caption keywords feed
search. Caption short + curiosity; 3–5 niche hashtags + one broad; hook in first 1s; native
trending sound low under the voice. Title/text overlay carries it (most watched with sound).

## Cross-platform reminders
- Each platform gets its OWN copy — don't paste the YouTube title into the Reel caption.
- Where a platform has both a title and a description/caption, the first line must differ
  from the title.
- Put every claim's link behind `[add link]` for the user to fill.
- End the file with a `## Sources` list of the trend articles you used.
